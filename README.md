# How to Use Addons
step 1 
- Type in the text box saying. 
```js
/functions uhc/setup
```
<br>

step 2
- Choose the team and color to play, then type in the text box saying.
```js
/functions game/start
```
<br>

step 3
- If we finish playing, type in the text box saying.
```js
/functions game/end
```
step 4
- Winner Command
```js
/functions rank/win
```
